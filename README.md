# Advanced Todo List

Todo list application with drag-and-drop, priority levels, due dates, and local storage persistence.

## Features

- Add, edit, delete todos
- Drag and drop reordering
- Priority levels (High, Medium, Low)
- Due dates and reminders
- Categories/tags
- Search and filter
- Local storage persistence
- Dark mode

## Tech Stack

- Vanilla JavaScript
- HTML5 Drag and Drop API
- LocalStorage API
- CSS3

## Usage

Open `index.html` in your browser.
